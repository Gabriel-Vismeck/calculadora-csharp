##
### Funcionalidades Principais:
* Adição;
* Subtração;
* Multiplição;
* Divisão;
* Design convidativo e intuitivo;
* Esquema de cores confortáveis aos olhos;
* Sistema de bloqueio e aviso ao clicar errado;
* Boa responsividade nos comandos.
